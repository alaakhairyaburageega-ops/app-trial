# Spoken English — نظام إدارة الحضور

يوفر تطبيق **Spoken English — نظام إدارة الحضور** لكل معلم مساحة عمل خاصة به: يسجّل دخوله، يستعرض لوحة تحكم بمؤشرات حضور متقدمة، ثم يتصفح المواد التي يُدرّسها، ويختار اليوم المحدد وفق الجدول الأسبوعي لكل مادة، ليسجّل حضور وغياب الطلاب المسجلين في تلك المادة تحديدًا.

يعتمد النظام على قاعدة بيانات **NeonDB (PostgreSQL)** بعلاقات مترابطة بين المعلمين والمواد والجداول الزمنية والطلاب، ويتم التواصل معها عبر **Netlify Functions** بدون خادم دائم (Serverless)، مما يتيح توسعًا مستقبليًا لدعم عدد غير محدود من المعلمين والمواد دون إعادة هيكلة النظام.
